# ⚖️ ДОФ Баланс — Telegram Bot

Телеграм-бот для автоматического мониторинга конвейерных весов, автоматического парсинга XLS-отчетов и анализа баланса доводно-обогатительной фабрики (ДОФ, Казахстан).

## 🚀 Основной функционал
* **Автоматический парсинг**: Принимает файлы отчетов `.xls` / `.xlsx`, считывает суточные данные по конвейерам и сохраняет их в локальную базу данных SQLite.
* **Суточная аналитика**: Вывод таблицы суточных балансов и объемов прохождения руды по ключевым конвейерам за выбранные периоды (день, неделя, месяц).
* **Контроль дублирующих весов**: Расчет расхождений между основными и дублирующими конвейерами (Кв4 vs 4Д, Кв3 vs 3Д).
* **Система алертов**: Автоматическое выявление нарушений технологических норм и критических отклонений баланса (Предупреждение: $\pm2\%$, Критично: $>5\%$).
* **AI-Ассистент** *(опционально)*: Интеграция с ИИ Claude (Anthropic) для глубокой диагностики отклонений, прогнозирования и выдачи метрологических рекомендаций.

## 🛠 Требования для локального запуска

Для работы бота на вашем компьютере должны быть установлены:
1. **Python 3.8+**
2. **VS Code** (или любая другая среда разработки)
3. Доступ в Telegram для регистрации бота через `@BotFather`.
